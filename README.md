## O que é DevOps?
DevOps é a combinação de "Desenvolvimento" (Dev) e "Operações" (Ops). Enquanto os desenvolvedores criam e aprimoram funcionalidades, a equipe de operações garante o desempenho e a estabilidade do sistema, o que pode gerar conflitos. O DevOps surgiu para integrar essas equipes, com profissionais atuando em todo o ciclo de vida do aplicativo, focando em práticas como infraestrutura como código, pipelines, virtualização e conteinerização.


## O que é SRE?
SRE (Site Reliability Engineering) é uma abordagem para garantir que sistemas sejam confiáveis, escaláveis e eficientes. O SRE utiliza práticas e ferramentas para monitorar, identificar e corrigir falhas ou anomalias no software.
